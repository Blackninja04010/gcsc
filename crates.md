# Crates and info on them

## Information
- Crate Multiplier does not effect keys

## Common

### Rewards
- 1/20 - Key (randomized 1-3)
  - 1/100 - Epic key
  - 1/50 - Rare key
  - 1/25 - Uncommon key
  - Else - Common key
- 1/10 - Multiplier (randomized 0.01-0.1)
  - 1/6 - Tradeshop Multiplier
  - 1/4 - Fishing Multiplier
  - Else - Crate Multiplier
- 1/5 - Stars
  - 1/1000 - 100K-250K
  - 1/500 - 50K-80K
  - Else - 250-1.25K
- Else - Money
  - 1/1000 - 250K-500K
  - 1/250 - 50K-100K
  - Else - 500-5K

  - 1/4 - Fishing Multiplier
  - Else - Crate Multiplier
- 1/5 - Stars

- Else - Money
  - 1/1000 - 500K-750K

  - Else - 1K-7.5K
